# lifeadmin-daily-note

Obsidian Daily Note template using **Templater**, **Tasks**, and **DataviewJS**.

## Goal

Replace slow live Dataview dashboard widgets (weather, workouts, weight, habits) with **static data baked in at note creation**, refreshable on button click.

## Plugins required

- [Templater](https://github.com/SilentVoid13/Templater)
- [Tasks](https://github.com/obsidian-tasks-group/obsidian-tasks)
- [Dataview](https://github.com/blacksmithgu/obsidian-dataview)

## Status

Work in progress — refactoring dashboard blocks from live queries to Templater-generated static buttons.
